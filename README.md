## ✨ 30+ module trong 1 Excel addin

### 📑 Hợp đồng & Thanh toán
- BoQ phân cấp 5 cấp độ + import từ Excel có sẵn
- Đợt thanh toán + lũy kế khối lượng nghiệm thu (% tạm ứng / giữ lại)
- Variation Order workflow (Draft → Submit → Approve)
- Bảo lãnh: Bid Bond / Performance / Advance Payment / Warranty

### 📅 Kế hoạch & Thực hiện
- Schedule + Gantt, Daily log + ảnh hiện trường
- Permits với cảnh báo hết hạn
- Approval Center, Submittal / RFI tracking

### 🏗️ Tài nguyên & Vật tư
- **Định mức 1776 Bộ Xây dựng VN** tích hợp sẵn
- Resource library (vật tư / nhân công / máy móc)
- Purchase Orders + Inbound + Usage tracking
- Apply norm template 1-click cho BoQ

### 💰 Tài chính
- Budget + % dự phòng, Cashflow projection
- Internal cost (chế độ Tổng thầu)
- Rate Book đơn giá đầu tư

### 📈 Báo cáo & KPI
- **Earned Value Management** (CPI, SPI, BCWS, BCWP, ACWP)
- **S-Curve** Plan vs Actual auto-render
- Dashboard tổng hợp xuất thẳng Excel sheet
- Subcontractor overview, Risk register

### 🛠 Vận hành
- VietQR thanh toán trong app — mua bản quyền không rời Excel
- Auto-update notification, Backup / Restore user data
- Giao diện song ngữ Tiếng Việt + English
- Theme Light / Dark

## 💻 Yêu cầu hệ thống

- **Windows 10** 1809+ (build 17763) hoặc Windows 11
- **Microsoft Excel 2016 trở lên** (32 hoặc 64-bit)
- **.NET 8 Desktop Runtime** — installer tự download nếu thiếu (~50 MB, một lần)
- Dung lượng trống: ~100 MB sau khi cài

## 🚀 Bắt đầu

1. Tải `DVDProjectMgr-Setup.exe` từ link bên trên.
2. Chạy installer (per-user, **không cần admin**).
3. Mở Excel — tab **DVDPM** xuất hiện trong ribbon.
4. Click **Mới** → tạo dự án đầu tiên trong vòng 5 phút.

Trial 30 ngày tự kích hoạt. Mua bản quyền khi cần: ribbon → **Mua bản quyền** → quét VietQR.
